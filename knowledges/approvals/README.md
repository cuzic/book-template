# 承認記録ディレクトリ

このディレクトリには、人間による承認の記録を保存します。

## 目的

- ワークフローの各マイルストーンで人間の承認を記録する
- 承認履歴を追跡可能にする
- 承認なしにフェーズを進めないことを保証する

## ファイル命名規則

```text
YYYY-MM-DD-milestone.md
```

例:

- `2025-12-15-outline.md` - 目次の承認
- `2025-12-20-chapter-01.md` - 第1章の承認
- `2025-12-25-publish.md` - 出版前の最終承認

## マイルストーンの種類

| マイルストーン | 説明 | 必須 |
|--------------|------|------|
| `outline` | 目次完成時の承認 | 必須 |
| `chapter-XX` | 各章完成時の承認 | オプション |
| `publish` | 出版前の最終承認 | 必須 |

## 担当

Author エージェントが人間の承認を得た後、`/book-approve` スキルで記録します。
