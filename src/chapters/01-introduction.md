# 第1章 はじめに

これはサンプルの章です。この本のテンプレートを使って、あなた自身の本を執筆してください。

## 概要

この章では、本書の目的と構成について説明します。

### このテンプレートの特徴

- **Markdown執筆**: シンプルなMarkdown形式で本を書くことができます
- **複数フォーマット対応**: HTML、XHTML、EPUBを自動生成
- **AI画像生成**: Gemini APIを使って挿絵を自動生成
- **日本語対応**: CJK（中日韓）言語に最適化

## 使い方

1. `src/chapters/`ディレクトリにMarkdownファイルを作成
2. `bun run build`でビルド
3. `dist/`ディレクトリに出力を確認

### コード例

```typescript
const message = "Hello, World!";
console.log(message);
```

このテンプレートを活用して、素晴らしい本を作りましょう。
