# book:outline - 目次生成・改善スキル

テーマから本の目次を生成、または既存の目次を改善します。

## 使い方

```
/book:outline <テーマ>     # テーマから新規目次を生成
/book:outline              # 既存の目次をknowledgesを考慮して改善
```

## 処理内容

### 新規目次生成（テーマ指定時）

1. 指定されたテーマを分析
2. 対象読者、本の目的を想定
3. 論理的な章構成を設計
4. `src/chapters/` に章ファイルのスケルトンを作成
5. 各章に見出し構造（H1, H2, H3）を含める

### 目次改善（テーマなし）

1. `knowledges/` ディレクトリの内容を読み込む
2. 現在の `src/chapters/` の目次構造を分析
3. knowledgesの知見を反映して目次を改善
4. 不足しているトピック、順序の改善を提案
5. ユーザー確認後、章ファイルを更新

## 出力フォーマット

各章ファイル（例：`src/chapters/01-introduction.md`）:

```markdown
# 第1章 はじめに

本章では〜について説明します。

## 1.1 背景

（執筆予定）

## 1.2 本書の目的

（執筆予定）

## 1.3 対象読者

（執筆予定）
```

## 注意事項

- ファイル名は `XX-slug.md` 形式（XXは2桁の番号）
- 章タイトルはH1で記述
- セクションはH2、サブセクションはH3
- textlintルールに準拠（ですます調、句点必須）
