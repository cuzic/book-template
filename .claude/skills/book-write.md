# book:write - 章執筆スキル

knowledgesを参照しながら章の本文を執筆します。

## 使い方

```
/book:write <章番号>        # 指定した章を執筆
/book:write 01              # 第1章を執筆
/book:write all             # 全章を順番に執筆
```

## 処理内容

1. 対象の章ファイル（`src/chapters/XX-*.md`）を読み込む
2. `knowledges/` から関連する知見を収集
3. `book.json` のメタデータを確認
4. 見出し構造に従って本文を執筆
5. 執筆後、lintチェックを実行
6. エラーがあれば修正

## 執筆ガイドライン

### 文体

- **ですます調**で統一
- 句点（。）で文を終える
- 読点（、）は1文に3個以内
- 1文は150文字以内

### 構成

- 各セクションの冒頭で概要を説明
- 具体例を交えて説明
- 必要に応じてコード例を挿入
- セクション末で要点をまとめる

### 参照

- knowledgesの情報を適切に引用
- 出典がある場合は明記
- 画像を使う場合は `images/` を参照

## 出力例

```markdown
# 第1章 はじめに

本章では、本書の目的と対象読者について説明します。

## 1.1 本書の目的

本書は〜を目的としています。具体的には、以下の3点を達成することを目指します。

1. 〜を理解する
2. 〜を実践できるようになる
3. 〜を応用できるようになる

### なぜこの本を書いたのか

筆者が〜に取り組む中で、〜という課題に直面しました。
この経験から、〜が必要だと考えるようになりました。

## 1.2 対象読者

本書は以下のような読者を想定しています。

- 〜の経験がある方
- 〜に興味がある方
- 〜を学びたい方

### 前提知識

本書を読むにあたり、以下の知識があると理解が深まります。

- 〜の基本概念
- 〜の使い方

## 1.3 本書の構成

本書は全X章で構成されています。

| 章 | タイトル | 内容 |
|----|---------|------|
| 1 | はじめに | 本章 |
| 2 | 基礎編 | 〜の基本を学ぶ |
| ... | ... | ... |

次章では、〜について詳しく説明します。
```

## 注意事項

- 執筆後は必ず `bun run lint` を実行
- 画像は `images/章番号-名前.png` 形式で参照
- コードブロックには言語を指定
- 長い章は複数セッションに分けて執筆
